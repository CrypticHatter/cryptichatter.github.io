My brand new blog using Hexo

2021-08-29

What is Hexo?

Welcome, This is my brand new blog developed using Hexo, Hexo is a simple, powerful and lightweight blogging famework. You can write your blog articles in markdown and other markup languages and hexo will generate static pages using them in seconds with slected marvelous themes.

I will describe how to start your blog using Hexo static page generate framework step by step in this blog article.

Installation

It’s only take few minutes to setup hexo for your blog site.

Requirements

  1. Node Js (Recommends 12.0 or higher)
  2. GIT (because we’re going to publish it using github pages)

If you already have above requirements you can skip this step and go strait to next step.

Install Node

Node js provide Setup for most operating systems.

Install Git
  1. For windows Installer
  2. For Mac Installer
  3. Linux (Ubuntu, Debian): sudo apt-get install git-core

Install Hexo
